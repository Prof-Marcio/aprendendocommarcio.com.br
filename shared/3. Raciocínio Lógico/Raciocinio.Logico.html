🛡️ MANTRA PARA A PROVA (RACIOCÍNIO LÓGICO)

Cuidado com a Condicional: No CEBRASPE, "Se p, então q" é equivalente a "~q → ~p" ou "~p ou q". Saiba trocar o conectivo com rapidez.

Negação não é antônimo: Negar "Todo" é "Pelo menos um... não". Negar "E" é usar "OU" com tudo negado (Leis de Morgan).

Estatística Básica: A mediana é o termo central após ordenar os dados (Rol). Se o desvio padrão é zero, todos os valores são iguais.

Análise de Gráficos: Não tente calcular tudo. Muitas questões de gráficos exigem apenas comparação visual e lógica de exclusão.
