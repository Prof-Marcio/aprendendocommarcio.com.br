Formação de Palavras

Processos de Derivação e Composição

🔎 Perfil da Banca — Fundação Carlos Chagas (FCC)

Estilo de cobrança: Questões conceituais + aplicação em palavras reais. Comum cobrar o processo dentro do contexto.

Tipo de raciocínio exigido: Identificar como a palavra foi formada; Diferenciar processos semelhantes (principal pegadinha).

Padrões recorrentes: derivação prefixal e sufixal; composição (justaposição × aglutinação); derivação imprópria.

Erros comuns: confundir prefixo com composição; confundir aglutinação com justaposição; não reconhecer derivação imprópria.

📚 NÍVEL 1 — VDG (Velho de Guerra)

📌 1. Conceito (o que é): Formação de palavras é o processo pelo qual novas palavras são criadas na língua portuguesa a partir de palavras já existentes, radicais ou afixos (prefixos e sufixos).

🎯 2. Aplicação / características: Serve para ampliar o vocabulário, criar novos significados e adaptar palavras a diferentes contextos. Muito cobrado para identificar o processo de formação.

🧩 3. Estrutura ou elementos principais:

🔹 1. Derivação (Forma palavras a partir de um único radical):

  • 📍 Derivação prefixal: Acréscimo de prefixo. Ex: feliz → infeliz
  • 📍 Derivação sufixal: Acréscimo de sufixo. Ex: feliz → felicidade
  • 📍 Derivação prefixal e sufixal: Prefixo + sufixo juntos. Ex: feliz → infelizmente
  • 📍 Derivação parassintética: Prefixo + sufixo simultâneos (não existe sem um deles). Ex: noite → anoitecer (não existe “noitecer” ou “anoite”).
  • 📍 Derivação imprópria: Mudança de classe gramatical sem alteração da forma. Ex: o jantar (verbo → substantivo).

🔹 2. Composição (Forma palavras com dois ou mais radicais):

  • 📍 Justaposição: Sem alteração das palavras. Ex: passatempo
  • 📍 Aglutinação: Com alteração fonética. Ex: plano + alto → planalto

⚙️ 4. Como funciona: A palavra é formada por radical (base) e afixos (modificam o sentido). A banca pede para identificar qual processo foi usado.

🎯 5. Como a FCC costuma cobrar: Identificação do tipo de derivação; diferença entre justaposição e aglutinação; reconhecimento de derivação imprópria.
Exemplo: A palavra “infelizmente” é formada por: Resposta: derivação prefixal e sufixal.

🟢 6. Exemplo simples: Palavra “anoitecer” ➡️ prefixo + radical + sufixo obrigatórios. Figura: derivação parassintética.

🧠 7. Dica de memorização (Pareto FCC):

  • 📌 prefixo → antes | 📌 sufixo → depois
  • 📌 justaposição → sem alteração | 📌 aglutinação → com alteração
  • 📌 parassíntese → depende dos dois
Resumo rápido: 🔹 Derivação → 1 radical | 🔹 Composição → + de 1 radical

⚫ NÍVEL 2 — Pegadinhas da banca (FCC)

  • 🔴 Prefixo + palavra = composição → 🔵 Errado. É derivação prefixal, não composição.
  • 🔴 Justaposição altera a forma das palavras → 🔵 Errado. Na justaposição não há alteração (ex: passatempo).
  • 🔴 Aglutinação mantém as palavras intactas → 🔵 Errado. Na aglutinação há alteração fonética (ex: planalto).
  • 🔴 Derivação parassintética permite retirar prefixo ou sufixo separadamente → 🔵 Errado. Na parassíntese, prefixo e sufixo são indispensáveis juntos.
  • 🔴 Derivação imprópria altera a estrutura da palavra → 🔵 Errado. Ela muda apenas a classe gramatical, não a forma.

⚫ NÍVEL 3 — Identificação rápida na prova (FCC)

🔴 prefixo antes da palavra → 🟢 “infeliz” → 🔵 Derivação prefixal
🔴 sufixo depois da palavra → 🟢 “felizdade” → 🔵 Derivação sufixal
🔴 prefixo + sufixo juntos → 🟢 “infelizmente” → 🔵 Derivação prefixal e sufixal
🔴 prefixo e sufixo obrigatórios → 🟢 “anoitecer” → 🔵 Derivação parassintética
🔴 duas palavras sem alteração → 🟢 “passa + tempo” → 🔵 Justaposição
🔴 duas palavras com alteração → 🟢 “plano + alto” → 🔵 Aglutinação
🔴 mudança de classe gramatical → 🟢 “O jantar foi ótimo” → 🔵 Derivação imprópria
